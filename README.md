# Portfolio App (React Native)

Aplicativo de portfólio mobile desenvolvido com React Native, com foco em apresentação profissional e reutilização como base para outros desenvolvedores.

---

## Visão Geral

Este projeto foi criado como um template reutilizável para desenvolvedores que desejam criar um aplicativo de portfólio próprio de forma rápida e simples.
